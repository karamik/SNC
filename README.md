# Звёздный удвоитель

Telegram бот с Mini App для игры "Удвоитель звёзд". 
Пользователи делают ставки в звёздах (100, 1000, 5000), каждую минуту происходит розыгрыш: 
40% игроков выигрывают x2, 50% получают кэшбек 10%, остальное — доход владельца.
Реферальная система, система уровней, ежедневные бонусы, магазин, турниры, админ-панель.

## Технологии
- Python 3.11
- aiogram 3.x
- FastAPI
- PostgreSQL
- Redis
- Docker / docker-compose

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте файл `.env` на основе `.env.example` и заполните своими данными
3. Запустите `docker-compose up -d`
4. Примените миграции (алхимия создаст таблицы автоматически при первом запуске)
5. Бот и API будут доступны:
   - Бот: запускается polling (или можно настроить webhook)
   - API: http://localhost:8000
   - Mini App: http://localhost:8000/static/index.html (после настройки nginx должен быть доступен по HTTPS)
